LeanDay 是一个每日减脂饮食安排 Web 端原型，适合用来展示减脂餐单、饮水打卡、热量分配和轻运动建议。

## 包含内容

- 三种减脂模式切换：均衡减脂、忙碌快手、高蛋白塑形
- 每日三餐两加餐时间轴
- 饮水打卡与完成度进度环
- 食材替换建议与运动提醒
- 导出今日计划文本

## 本地打开

直接用浏览器打开 `index.html` 即可。

## 文件说明

- `index.html`：页面结构
- `style.css`：视觉样式与响应式布局
- `script.js`：页面交互与计划数据
